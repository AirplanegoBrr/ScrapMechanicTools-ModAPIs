### sci.out

```lua
local active = true

-- Numbers
if active then
    sci.out(1)
else
    sci.out(0)
end

-- Boolean
if active then
    sci.out(true)
else
    sci.out(false)
end
```


Sets the self.interactable.active

<strong>Arguments:</strong> <br></br>

- <code>Output</code> [<strong> number | boolean </strong>]: The value for self.interactable.active
